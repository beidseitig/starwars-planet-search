# Contexto
Aplicação para demonstrar uso de Hook's padroes e personalizados e ContextAPI

## Técnologias usadas

Front-end:
> Desenvolvido usando: React, ContextAPI, CSS3, HTML5, ES6

## Instalando Dependências

> Frontend
```bash
npm install
``` 
## Executando aplicação

* Para rodar o front-end:

  ```
    cd src/ && npm start
  ```
